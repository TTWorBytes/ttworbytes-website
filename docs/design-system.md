TTWORBYTES Design System v1.0

Author: Neil Amess

Website: TTWORBYTES

Version: 1.0

Purpose

Every page should feel like it belongs to the same world, whether it is promoting a game, a book, a character or a piece of news. The design should be modern, cinematic and welcoming, with storytelling at its heart.

1. Brand Personality

    TTWORBYTES should feel:

    ⭐ Imaginative

    ⭐ Family-friendly

    ⭐ Professional

    ⭐ Warm

    ⭐ Adventurous

    ⭐ Modern

    Never corporate.

    Never cluttered.

    Never childish.

2. Design Principles

    Whenever we design something, ask:

    Would this make someone want to explore another page?

    Everything should encourage curiosity.

    The website should invite exploration.

3. Colour Palette
    Primary Background
    #0B1020

    Deep Space Blue

    Secondary Background
    #11192E

    Panels and sections

    Gold Accent
    #D6A94A

    Used sparingly.

    Buttons.

    Links.

    Highlights.

    Never huge blocks of gold.

    Primary Text
    #FFFFFF
    Secondary Text
    rgba(255,255,255,0.78)
    Borders
    rgba(255,255,255,0.08)
    Success Accent
    #61D68B

    Only if needed.

4. Typography

    The typography should be easy to read and never cramped.

    H1

    Large.

    Bold.

    Usually one or two lines.

    H2

    Clear.

    Generous spacing above.

    H3

    Used inside cards.

    Paragraphs

    Maximum width:

    70 characters

    Never let text stretch across the entire screen.

5. Page Width

    Every page should follow:

    Maximum Content Width

    1200px

    Glass panels:

    900–1100px

    depending on content.

6. Spacing

    Consistency is more important than exact measurements.

    Major sections

    100px

    between them.

    Inside panels

    40px

    Card gaps

    30px

    Paragraph spacing

    20px
7. Glass Panels

    Every glass panel should feel identical.

    Rounded corners

    20px

    Border

    Subtle.

    Background

    Semi-transparent.

    Blur

    Soft.

    Shadow

    Gentle.

    Never harsh.

8. Buttons

    Every button should look like it belongs to TTWORBYTES.

    Rounded pill shape.

    Outline by default.

    Gold on hover.

    Quick transition.

    Never oversized.

9. Cards

    Whether it's:

    Book

    Character

    Game

    News

    Timeline

    they should all follow the same philosophy.

    Consistent padding.

    Consistent radius.

    Consistent shadows.

    Consistent hover movement.

10. Images

    Book covers

    Same radius.

    Portraits

    Same size whenever possible.

    Transparent PNGs preferred for characters.

    Artwork should always be high quality.

    Never stretch images.

11. Animation

    Animations should support the content, not distract from it.

    Cards

    Lift slightly.

    Buttons

    Glow softly.

    Sections

    Fade in gently.

    Navigation

    Smooth.

    Never spin.

    Never bounce.

    Never flash.

12. Navigation

    Navigation should always answer:

    Where am I?

    Where can I go next?

    The current page should always be obvious.

13. Character Pages

    Every character page should follow the same structure.

    Hero
    Summary
    Quote
    About
    Four detail cards
    Appearance
    Relationships
    Connections
    Closing section
14. Book Pages

    Every book page should include:

    Hero

    Synopsis

    Why you'll enjoy it

    Characters

    Gallery

    Series position

    Next book

15. Timeline Pages

    Every timeline entry should contain:

    Book number

    Cover

    Description

    Main event

    Link

    Consistent layout.

16. Writing Style

    Everything on the website should sound like:

    Someone inviting you into a story.

    Not someone selling you a product.

    Use:

    Warm

    Friendly

    Confident

    Curious

    Optimistic

    British English

    Avoid:

    Marketing clichés

    Excessive exclamation marks

    Long paragraphs

    Spoilers

17. Accessibility

    Aim for:

    Good contrast

    Readable text

    Descriptive image alt text

    Keyboard-friendly navigation

    Responsive layouts

    The goal is that anyone can comfortably explore the site.

18. Performance

    Every page should load quickly.

    Use appropriately sized images.

    Avoid unnecessary scripts.

    Keep animations lightweight.

    A fast site feels more professional.

19. Future Sections

    Any new section—whether it's Ke'Valien, Games, News, or future book series—should reuse the same design language. The content can change, but the visitor should immediately recognise that they're still in the TTWORBYTES world.

20. The TTWORBYTES Test

    Before you add or change anything, ask these five questions:

    Does it look like it belongs on TTWORBYTES?
    Is it simple to understand?
    Will it encourage visitors to explore another page?
    Is it consistent with the rest of the site?
    Does it support storytelling rather than distract from it?

    If the answer to all five is yes, it's probably the right design choice.
