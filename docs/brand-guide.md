# TTWORBYTES Brand Guide

## Core Idea

TTWORBYTES creates stories, games and worlds built around the people who matter most.

## Brand Feeling

Quiet confidence.
Cinematic.
Emotional.
Curious.
Warm.
Inviting.
Never cluttered.

## Main Message

Every adventure begins with the people who matter most.

## Visual Style

- Dark cinematic backgrounds
- Warm gold highlights
- Soft blue atmosphere
- Glass panels
- Generous negative space
- Gentle movement
- Calm, premium layout

## Colours

Background: `#0B1020`

Gold: `#D6A94A`

Blue: `#4FA8FF`

Main text: `#FFFFFF`

Secondary text: `#C7D0E0`

## Typography

Headings: Cinzel

Body text: Inter

## Design Rules

1. Every section should have one clear purpose.
2. Do not clutter the negative space.
3. Use movement only when it adds feeling.
4. Every world should feel distinct but still belong to TTWORBYTES.
5. Reuse components wherever possible.
6. The website should feel like entering a story, not browsing a catalogue.
